# 风水实用指南 - 项目规格

## 核心功能
用户选择场景（工位/客厅/卧室等）→ 选择自己的位置/朝向 → 选择想提升的运势 → 得到具体的物品+方位建议

## 页面结构

### 1. 场景选择
- 🏠 客厅
- 🛏️ 卧室
- 💼 工位
- 🍳 厨房
- 🚪 大门

### 2. 方位判断（核心）
用户用罗盘或直觉选择自己的位置相对于房间的方位：
- 坐北朝南 / 坐南朝北 / 坐东朝西 / 坐西朝东
- 或者用九宫格：西北 正北 东北 / 西 正中 东 / 西南 东南 南

### 3. 运势选择
- 💰 财运
- 💼 事业
- ❤️ 桃花
- 📚 学业
- 💪 健康

### 4. 输出结果
具体的建议卡片，包含：
- 推荐的物品（如：绿植、水晶、灯光、书籍等）
- 具体摆放方位（东/东南/南等）
- 为什么有效（简短原理）
- 摆放小技巧

## 核心数据库

### 工位建议示例（坐北朝南为例）
| 想要提升 | 物品 | 方位 | 原理 |
|---------|------|------|------|
| 财运 | 小水缸/流水摆件 | 正东 | 木生火，火生土，土生金 |
| 事业 | 绿萝/小型盆栽 | 东南 | 文昌位催事业 |
| 桃花 | 粉色水晶/干花 | 正南 | 离卦主桃花 |
| 健康 | 小型植物/医药箱 | 正北 | 坎卦主健康 |

### 客厅建议
- 沙发朝东：财位在东南，适合放招财植物
- 电视墙朝西：适合放金属装饰平衡

## 技术实现
- 单HTML文件
- 原生JS，无框架
- 八卦罗盘SVG动画
- 流畅的转场动画
- 响应式设计
